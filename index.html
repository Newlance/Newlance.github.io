Website Design and Development

Website development is an interesting middle ground for anyone with a programming mindset. Web design requires more than just understanding of programming languages, it also requires a knack for visual design and color. Some would argue these skills are becoming more important than language, as web development grows closer to being an automated process. While this may be true, there is still plenty of room for Web Programmers. The fact is, someone must be developing these "easy-to-use web design tools" that you see, such as WordPress. This is certainly where I would feel most comfortable; however, comfort is a leisure the programmer can seldom afford. The programmer must adapt to overcome new arising obstacles.
